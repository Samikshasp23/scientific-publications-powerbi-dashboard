# Scientific Publications Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard built to analyze scientific publication data and uncover trends related to research output, author contributions, and conference distribution. The dashboard focuses on transforming raw data into meaningful insights through interactive visualizations and KPIs.

## Key Insights

* Publication trends across multiple years
* Author contribution analysis
* Conference-wise comparison
* Total publications KPI
* Average authors per paper (collaboration insights)
* Top contributing authors
* Interactive drill-through for detailed analysis
* Dynamic filtering using slicers

## Tools & Skills Used

* Power BI
* Power Query
* DAX
* Data Modeling
* Data Visualization

## Dashboard Features

* Interactive slicers for filtering by year and conference
* KPI cards for quick insights
* Author performance analysis
* Publication trend analysis
* Clean and user-friendly layout
* Navigation buttons for multiple pages
